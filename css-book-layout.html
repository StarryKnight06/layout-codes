CSS Book Layout

This code is based on Repth's Open Book Tutorial! I used it for my scrapbook webpage and a couple other places.

The original tutorial used the float property for the layout which conflicted with what I wanted to add. So, I reworked the code with CSS grid instead to better fit my needs. The layout is also mobile friendly and should look okay on different screen sizes!
